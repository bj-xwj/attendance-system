# UI 组件说明

## 概述
本系统采用 Next.js 14 的 App Router 架构，使用 Tailwind CSS 进行样式设计，实现了响应式的用户界面。

## 页面结构

### 公共组件
- **`layout.tsx`**: 根布局，包含 ToastProvider 和全局样式
- **`sidebar.tsx`**: 侧边栏导航菜单，根据用户角色显示不同选项

### 布局路由
系统采用嵌套路由结构：
```
app/
├── (dashboard)/          # 仪表盘布局
│   ├── employee/         # 员工端页面
│   ├── hr/               # 人事端页面
│   └── admin/            # 管理员页面
├── login/                # 登录页
└── register/             # 注册页
```

## 员工端页面 (employee)

### `/employee/page.tsx` - 工作台
**功能**: 
- 显示今日打卡状态和出勤率
- 展示本月考勤统计卡片
- 列出最近的考勤记录和待审批请假
- 提供快捷操作入口（打卡、请假等）

### `/employee/attendance/page.tsx` - 打卡签到
**功能**:
- 实时显示当前时间
- 上下班打卡按钮
- 显示打卡规则（上班时间、迟到阈值等）
- 记录当前工时

### `/employee/leave/page.tsx` - 请假申请
**功能**:
- 新建请假申请表单（类型、日期、原因）
- 显示请假统计（总申请、待审批、已通过等）
- 列出我的请假记录

### `/employee/overtime/page.tsx` - 加班申请
**功能**:
- 新建加班申请表单（日期、时间段、原因）
- 显示加班统计
- 列出我的加班记录

### `/employee/records/page.tsx` - 考勤记录
**功能**:
- 按月份筛选考勤记录
- 显示月度统计（出勤天数、总工时等）
- 列出每日明细（打卡时间、状态等）

## 人事端页面 (hr)

### `/hr/page.tsx` - 人事工作台
**功能**:
- 显示待审批事项总数
- 展示本月异常考勤数量
- 统计员工总数
- 列出待审批请假和加班申请
- 提供快捷操作入口

### `/hr/approvals/page.tsx` - 审批管理
**功能**:
- 选项卡切换（请假审批/加班审批）
- 待审批列表（支持批量通过）
- 状态筛选（全部/待审批/已通过/已拒绝）
- 审批意见输入框

### `/hr/employees/page.tsx` - 员工管理
**功能**:
- 添加/编辑/删除员工
- 员工信息表（工号、姓名、部门、角色等）
- 搜索和筛选功能
- 角色权限管理

### `/hr/corrections/page.tsx` - 考勤修正
**功能**:
- 新增考勤修正（补录或修改记录）
- 修正类型选择（补录上下班打卡、状态修正、补录整条记录）
- 修正原因说明
- 修正记录查询

### `/hr/records/page.tsx` - 考勤记录
**功能**:
- 按月份和员工筛选
- 查看所有员工的考勤记录
- 显示详细打卡时间

### `/hr/reports/page.tsx` - 统计报表
**功能**:
- 月度考勤数据汇总
- 柱状图展示员工考勤分布
- 饼图展示考勤状态占比
- 导出 CSV 功能
- 详细表格（支持部门和搜索筛选）

## 管理员端页面 (admin)

### `/admin/page.tsx` - 管理员工作台
**功能**:
- 系统概览（员工总数、考勤记录等）
- 待审批事项汇总
- 最近考勤修正记录
- 快捷操作入口

### `/admin/users/page.tsx` - 用户管理
**功能**:
- 查看所有用户信息
- 修改用户角色权限
- 角色变更即时生效

### `/admin/settings/page.tsx` - 系统设置
**功能**:
- 班次名称配置
- 上下班时间设置
- 迟到/早退阈值设置
- 设置保存与反馈

## 工具函数

### `lib/utils.ts`
提供常用的工具函数：
- `formatDate`, `formatTime`, `formatDateTime`: 时间格式化
- `getStatusLabel`, `getStatusColor`: 状态标签和颜色映射
- `getLeaveTypeLabel`: 请假类型标签
- `getRoleLabel`: 角色标签
- `getCurrentDate`, `getCurrentMonth`: 获取当前日期

## 样式规范
- 使用 Tailwind CSS 类名进行样式设计
- 主要颜色：primary (蓝色)、success (绿色)、warning (黄色)、danger (红色)
- 响应式设计，适配移动设备
- 卡片式布局，圆角设计
- 图标使用 lucide-react 库
- 图表使用 recharts 库

## 交互特性
- 表单验证（必填项、邮箱格式等）
- 实时时间显示
- Toast 提示消息
- 模态框（用于添加/编辑）
- 批量操作（批量审批）
- 数据导出功能
- 筛选和搜索功能
- 分页加载（部分页面）
- 状态实时更新
